# Backend / 后端

This directory will contain the backend services for the whichWitch platform.

这个目录将包含 whichWitch 平台的后端服务。

## Planned Features / 计划功能

- Event indexing service (Supabase Edge Functions)
- Metadata storage API
- Work data synchronization
- User authentication

## Tech Stack / 技术栈

- Supabase
- Node.js
- TypeScript
