# Frontend / UI / 前端

This directory will contain the frontend application for the whichWitch platform.

这个目录将包含 whichWitch 平台的前端应用。

## Planned Features / 计划功能

- Wallet connection (MetaMask)
- Work registration interface
- Authorization request flow
- Creation tree visualization
- Balance and withdrawal UI
- Tipping functionality

## Tech Stack / 技术栈

- Next.js
- React
- TypeScript
- ethers.js
- wagmi
- RainbowKit
- TailwindCSS
